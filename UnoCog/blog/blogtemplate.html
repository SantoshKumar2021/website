Announces CodeSpaces, Discussions, Code Scanning and much more at Satellite 2020

Satellite 2020 is GitHubs annual product and community conference featuring GitHub product experts, software leaders. This years event happened to be virtual and free. It happened yesterday May 6,2020. Follow the blog to know brief of some of the major announcements.

Major Announcements at Satellite 2020

Codespaces:

A complete dev environment within GitHub that lets you contribute immediately.This means you dont need to install a dev setup and can get the VS Code like environment right from your browser. (Start coding in a second) This is currently in limited public beta and you can subscribe to get early access for choice of your available programming language today by clicking below link. Expected to have more advanced features relating to builds in a pricing model.

CodeSpaces..

GitHub Discussions:

A new way for software communities to collaborate outside the codebase. Currently GitHub allows conversations in a thread based forum for issues and pull requests. Now they are taking one step further to allow conversations to be separately tracked to create and promote community knowledge base.(Maintaining FAQs and collaborative development easily accessible) This will be available soon in beta for public repositories

Code scanning and secret scanning:

Helping communities on GitHub produce and consume more secure code. Code scanning is now part of native GitHub experience and now every code 'git push' code is scanned for potential vulnerabilities results displayed in pull request. Secret scanning is now available for all private repositories.

GitHub Private Instances:

Collaboration even for stringently regulated customers A lot of companies are very secure especially enterprises who have very strict security requirements. They are still dependent on GitHub communities but are reluctant because of the same security reasons. So GitHub has now introduced private instances that provides enhanced security, compliance, and policy features including bring-your-own-key encryption, backup archiving, and compliance with regional data sovereignty requirements.
